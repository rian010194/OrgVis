Theme Saving Not Working
Active

When editing theme settings (colors, organization name, description, logo), the "Save Theme" button does not save changes to the database. This affects all theme customization features.

Workaround:

• Try refreshing the page and editing theme again

• Check browser console for error messages

• Ensure you have a stable internet connection

• Try logging out and back in

Map Focus Issues
Investigating

When switching from tree view to map view, the map may not focus on the correct node location. The camera sometimes moves to unexpected positions.

Workaround:

• Use manual pan/zoom to navigate to the desired location

• Switch back to tree view and select the node again

• Try refreshing the page

Organization Switching During Theme Save
Investigating

Sometimes when saving theme changes, the application may switch to a different organization unexpectedly. This appears to be related to database queries.

Workaround:

• Use the logout button to return to the landing page

• Select your organization again from the landing page

• Avoid making multiple rapid theme changes
